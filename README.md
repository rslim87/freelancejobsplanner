# README

This app is used by freelancers to keep track of their jobs and clients.

To use this app:

1. Clone or download the app

2. Run 'bundle install'

3. Run 'rake db:migrate'

3. Run 'rails s'

4. Open up your browser and visit http://localhost:3000
